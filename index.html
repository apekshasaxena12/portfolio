AI · RAG · Full-Stack
Codebase Understanding Assistant

A code intelligence tool that accepts any GitHub repo URL and answers natural language questions about the codebase — like GitHub Copilot, Cursor, or Sourcegraph.

  • Chunks by AST function boundaries (Python) or overlapping windows (16+ languages); stores 384-dim Jina AI embeddings in pgvector
  • Hybrid retrieval: dense vector search + BM25 full-text search fused via Reciprocal Rank Fusion for precision
  • Interactive D3.js knowledge graph, dependency graph, and Mermaid.js architecture diagrams with per-node Q&A
  • AST-extracted symbols injected into LLM context to minimize hallucination on code queries

minideploy — Self-Hosted Deployment Platform

A minimal Vercel/Netlify alternative — users provide a GitHub repo, build command, and output directory. The platform clones, security-scans, builds, uploads to Cloudflare R2, and serves at <slug>.minideploy.xyz.

  • End-to-end deployment pipeline: GitHub Actions CI → security scan → build → upload to Cloudflare R2 → serve
  • Build errors and scan failures explained in plain English by Gemini Flash LLM
